# Network-Intrusion-Detection-System-using-Multiclassification
A robust Network Intrusion Detection System (NIDS) using LightGBM for Multi-Class Classification on the UNSW-NB15 dataset. Achieves 83.88% accuracy by leveraging GOSS &amp; EFB for anomaly detection against 9 attack families. Includes a Hybrid Rule Engine and a Streamlit GUI for real-time SOC deployment.
